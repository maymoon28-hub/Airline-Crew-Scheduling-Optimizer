# Airline Crew Scheduling Optimizer

## Project Overview
Developed a crew scheduling simulation to assign pilots and cabin crew to a weekly flight schedule, using data, Python, and Excel to build a simple decision-support tool for operational planning.

## Problem Statement
Airlines need to ensure every scheduled flight is staffed with the right mix of qualified crew while respecting operational constraints, such as:
- **Maximum weekly hours** per crew member
- **Minimum rest periods** between duties
- **Role/qualification requirements** (pilots vs cabin crew)
- **Coverage targets** (no unstaffed flights)

This project models a simplified version of that problem and shows how data-driven logic, KPIs, and reporting can support scheduling decisions.

## Solution Approach
The solution consists of a Python script that:
- Reads a **flight schedule** and **crew roster** from CSV files
- Uses a **greedy assignment algorithm** to allocate available crew to flights under weekly-hour limits
- Computes **key performance indicators (KPIs)** such as flight coverage, crew utilization, and total assignments
- Exports results to CSV for visualization and further analysis in Excel

This setup demonstrates skills in data modeling, basic optimization logic, and building lightweight analytics tools around operational workflows.
